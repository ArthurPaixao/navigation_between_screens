# My Compose Navigation

## Como abrir no Android Studio

1. Abra o Android Studio.
2. Selecione **File > Open**.
3. Navegue até `C:\Users\DELL\my-compose-navigation` e clique em **OK**.
4. Aguarde o Gradle sincronizar o projeto.

## Configuração do Emulador

1. No Android Studio, vá para **Tools > Device Manager**.
2. Clique em **Create Device**.
3. Escolha um hardware (ex: Pixel 7) e clique em **Next**.
4. Escolha uma imagem do sistema (ex: Android 14 / API 34) e clique em **Next**.
5. Clique em **Finish**.
6. Para rodar o app, selecione o emulador criado na barra superior e clique no botão **Run** (triângulo verde).
